# TripleTen web_project_expenses

## Descripción

Este archivo contiene la lógica principal del **Gestor de Gastos**. Define las variables centrales del presupuesto y los gastos, calcula estadísticas como el _gasto promedio_, el _saldo disponible_ y el _total por categoría_, y determina qué categoría concentra el mayor gasto. También gestiona la adición de nuevos gastos al sistema, actualizando los totales en tiempo real.
